# Advanced Memory Cleaner Pro

![Memory Cleaner](https://img.shields.io/badge/Version-1.1.0-blue.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)
![Python](https://img.shields.io/badge/Python-3.8+-green.svg)

一款专业的 Windows 内存优化工具，采用现代化的 Python 技术开发，提供直观的内存管理和清理功能。

## 🌟 主要特性

### 实时内存监控
- 精确显示物理内存、虚拟内存和系统工作集使用情况
- 四种内存类型实时监控：物理内存、虚拟内存、系统工作集、进程工作集
- 可视化内存使用趋势图表

### 智能内存清理
- **工作集清理** - 通过 EmptyWorkingSet API 清理进程工作集
- **系统工作集清理** - 优化系统工作集和备用列表
- **备用列表清理** - 清理系统缓存，提升内存利用效率
- **虚拟内存优化** - 通过垃圾回收机制优化内存分配

### 自动化功能
- **自动清理机制** - 设置内存阈值，自动触发清理操作
- **内存告警系统** - 自定义告警阈值，实时监控内存状态
- **智能冷却时间** - 防止频繁清理，优化系统性能

### 用户体验
- **现代化界面** - 基于 CustomTkinter 开发的深色主题界面
- **系统托盘集成** - 后台运行不干扰用户，实时显示内存状态
- **历史数据图表** - 点击内存卡片查看详细使用趋势
- **Windows 11 通知** - 优雅的右下角通知提醒

## 🚀 系统要求

- **操作系统**: Windows 10 / 11（推荐）
- **权限要求**: 推荐以管理员权限运行以获得最佳效果
- **运行环境**: .NET Framework 4.5 或更高版本

## ❓ 常见问题

### Q: 为什么需要管理员权限？
A: 管理员权限允许程序访问系统级内存管理功能，提供更彻底的内存清理效果。

### Q: 自动清理会影响系统性能吗？
A: 不会。程序具有智能冷却时间机制，避免频繁清理，确保系统稳定性。

### Q: 程序会收集用户数据吗？
A: 不会。所有数据都在本地处理，不会上传任何用户信息。

### Q: 如何验证清理效果？
A: 清理完成后，程序会显示释放的内存大小，并记录在操作日志中。

## 🐛 故障排除

### 清理效果不明显
- 确保以管理员权限运行程序
- 检查是否启用了所有清理选项
- 系统内存可能已经优化，无需进一步清理

### 程序无法启动
- 确认系统满足要求（Windows 10/11）
- 检查是否被安全软件拦截
- 尝试重新下载最新版本

### 更新检查失败
- 检查网络连接是否正常
- 确认防火墙未阻止程序访问网络
- 手动访问 GitHub Releases 页面下载更新
