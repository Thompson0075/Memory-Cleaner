Advanced Memory Cleaner Pro 使用说明

欢迎使用 Advanced Memory Cleaner Pro！

系统要求：
- Windows 10/11 操作系统
- 推荐以管理员权限运行以获得最佳效果

主要功能：
1. 实时内存监控
2. 多种内存清理选项
3. 自动清理功能
4. 历史数据图表显示
5. 系统托盘监控

使用建议：
- 首次使用时，建议以管理员权限运行程序
- 可以根据需要调整自动清理的阈值
- 程序可以最小化到系统托盘，不影响正常使用
